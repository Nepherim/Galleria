!! Update Summary {$:Version} [[#updates]]
This upgrade incorporates the latest version of the [[http://galleria.aino.se/ | Galleria]] javascript library. It's a significant change, with easier setup, multiple carousels per page, less options to worry about, and optional integration with Lightbox. With simplification we did loose some of the sample skins, and also lost the ability to display vertical carousels.

If you are upgrading, make sure you:
* Remove all current [@$galleria[]@] [[#options | options]], except [@list@], [@height@], and [@width@].
* [@$galleria['list']@] should now specify the container for the list of images, not the list itself. So, prrobably a change to remove the [@UL@:
** From: [@$galleria['list'@] = 'MY_DIV UL';
** To: [@$galleria['list'@] = 'MY_DIV';
* Replaces the use of Skins, with a theme concept -- only one sample is provided.
* You may need to either add global [[#options | width and height options]], or accept a default of 500px.

!! Installation [[#installation]]%h2%
To install:
* Download either the [[http://github.com/Nepherim/{$Name}/zipball/master | ZIP]] or [[http://github.com/Nepherim/{$Name}/tarball/master | TAR]], and extract all files in the [@pub@] and the [@cookbook@] directory  into your [@pub@] and [@cookbook@] directories respectively. You should end up with a [@galleria@] directory in both [@cookbook@] and [@pub@].
  After extracting the files you should have:
>>box<<
[@
 cookbook/galleria/
      |- galleria.php
      |- galleria.txt
      |
 pub/galleria/
      |- galleria-1.2.3.js
      |- galleria-1.2.3.min.js
      |- LICENSE
      |
      |- /themes/
                |- classic
                |      |- SEVERAL THEME FILES
@]
>><<
* Add this line to your [@config.php@]:
>>box<<
[@include_once("$FarmD/cookbook/galleria/galleria.php");@]
>><<

!! Usage
Simply create a list of images, and then point galleria to that list using an xpath reference to the list container, or the list itself. Many more examples on [[http://wiki.solidgone.org/Galleria | solidgone.org]].

[@
(:div id="demo1":)
* Attach:P5123230.jpg
* Attach:P5123234.jpg
* Attach:P5123241.jpg
(:divend:)
(:galleria list="#demo1")
@]


!! Options [[#options]]%h2%
The main option you need to set is [@list@] which is the xpath to the image list. In addition you can use any of the [[http://galleria.aino.se/docs/1.2/options/ | options available to Galleria]]:
* '''list''': The full xpath to the image list container. Use ".pmGalleryWrapper" with pmGallery.
* '''width/height''': The maximum width/height to display the main image.

All options can either set in your config.php, where you know they will apply site wide, or at each [@ (:galleria:)@] directive.
* In order to set opttions in [@config.php@] use this format:
>>box<<
  [@ $galleria['option_name'] = 'value';@]
>><<
* In order to set options with the markup directive use this format -- note that quotes are optional, but '''must''' be used if the value contains spaces.
>>box<<
  [@ (:galleria option_name="value" :) @]
>><<


!! Changing Themes [[#themes]]%h3%
Galleria comes with a single sample theme, [@classic@]. If you load or [[http://galleria.aino.se/docs/1.2/themes/creating_a_theme/ | create additional themes]], update the $HTMLHeader with the apprropriate path:
>>box<<
[@$HTMLHeaderFmt['galleria-theme'] = '<script type="text/javascript">Galleria.loadTheme("'. $PubDirUrl. '/galleria/themes/classic/galleria.classic.min.js");</script>';@]
>><<
